## Dependencies

* [ino tools](http://inotool.org/)

## Building & Installing

* Build the project
```
ino build
```
* Upload to the Arduino
```
ino upload
```
## Flash arduino boot loader

| New Arduino | Working Arduino |
| --- | --- |
| pin 1 | D12 |
| pin 2 | 5V  |
| pin 3 | D13 |
| pin 4 | D11 |
| pin 5 | D10 |
| pin 6 | GND |
