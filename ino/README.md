## How to build

* Get [ino tools](http://inotool.org/)
* Build the project
```
ino build
```
* Upload to the arduino
```
ino upload
```

## Flash arduino boot lader

| New Arduino | Working Arduino |
--- | --- | 
| pin 1 | D12 |
| pin 2 | 5V  |
| pin 3 | D13 |
| pin 4 | D11 |
| pin 5 | D10 |
| pin 6 | GND |
